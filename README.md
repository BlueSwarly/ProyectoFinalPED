# ProyectoFinalPED
Proyecto final de programacion de estructuras dinamicas

**Caracteristicas del programa**  

**Menú Principal**
Permite el acceso a las diversas opciones de juego y a los submenús organizados para la gestión de las partidas.

***Inscripción de Participantes** 
Realiza la solicitud del nombre o de un carácter identificativo para cada integrante de la competencia.

**Ayuda**
Proporciona la explicación detallada sobre el manejo del juego en todas sus etapas, así como el propósito de cada opción dentro de los menús de la aplicación.
